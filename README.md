## 个人博客系统
#### 需求分析
```
1.用户管理
    注册,登录,增加用户,删除用户,搜索用户
2.安全设置  spring security
    角色授权,权限设置
3.博客管理
    发表博客,编辑博客,删除博客,博客分类,设置标签,上传图片,模糊查询,最热排序,最新排序,阅读量统计
4.评论管理
   发表评论,删除评论,评论量统计
5.点赞管理
   点赞,取消点赞,点赞量统计
6.分类管理
   创建分类,编辑分类,删除分类,按分类查询.
7.标签管理
   创建标签,删除标签,按标签查询
8.首页搜索  elastic search
   全文检索,最新文章,最热文章,热门标签,热门用户,热门文章,最新发布
9. .....
```
